# Product Requirements Document (PRD) - AarogyaAid

## Overview
AarogyaAid is an AI-powered insurance recommendation platform designed to help users find the best insurance plans tailored to their needs.

## Features
- AI chatbots for insurance recommendations
- RAG using vector databases for accurate policy information retrieval
- Authentication and User Profiles
- Policy Management

## Tech Stack
- **Frontend**: React (Vite) + TailwindCSS
- **Backend**: FastAPI + Python
- **Database**: PostgreSQL
- **Vector Store**: ChromaDB
- **LLM**: Google GenAI / LangChain
