# AarogyaAid

An AI-powered insurance recommendation platform.

## Structure

- `/frontend`: React.js + Vite frontend.
- `/backend`: FastAPI Python backend.

## Setup

1. Copy `.env.example` to `.env` and fill in the values.
2. Navigate to `backend/` and set up the Python virtual environment, then install requirements.
3. Navigate to `frontend/` and install node modules.
